Interactive Incident Response Training Game
An interactive web-based game for practicing incident response scenarios.
-
Frontend: HTML, CSS, JavaScript, Phaser.js
-
Backend: Python, Flask
-
Database: SQLite
##
Setup
1.
Clone repository
2.
Install dependencies:
`pip install -r backend/requirements.txt`
3.
Run backend:
`python3 backend/app.py`
4.
Open frontend:
`http://localhost:8000`
